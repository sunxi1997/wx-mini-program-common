# common

```

│   │ 
│   ├── debounce.js     // 函数防抖
│   ├── throttle.js     // 函数节流
│   ├── wxPayment.js    // 封装微信支付
│   └── index.js        // 通用 js 集成文件
│
└── typings             // 能让 webstorm 提示


```
